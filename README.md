## 🎧 What Muji Does

Muji helps you stay in flow while you code. It can:

- Send TTS voice alerts
- Use ElevenLabs voices for clearer spoken prompts
- Run a Pomodoro timer for focused sessions

## 🔊 Audio and Voice Options

Muji works with music and speech tools so you can shape your work space.

- Ambient background music for long coding sessions

## 🕒 Pomodoro Timer

The Pomodoro timer helps you break work into short blocks. A common setup is 25 minutes of focus and 5 minutes of rest. You can change this to match your pace.

Use Pomodoro mode if you:

- Lose track of time
- Work too long without breaks
- Want a simple focus cycle
- Need a clean stop point between tasks

## 🧘 Wellness Alerts

Muji can show wellness reminders during long sessions. These prompts help you rest your eyes, stretch, drink water, or take a short walk.

Use wellness alerts if you spend many hours at the screen. They can help you keep a steady pace through the day.

## 💤 Idle Detection

Idle detection checks when you stop using the keyboard or mouse. Muji can use that signal to pause music, change alerts, or note that you stepped away.

This is useful when you:

- Leave your desk for a break
- Get pulled into a call
- Want your session tools to respond to pauses
